About
What is FLIPSWITCH?

FLIPSWITCH is a Minecraft: Java Edition datapack. It lets players code their own minigames in their own single player world. No mods, no launchers. Drop it in your datapacks folder and it works.

Every flip changes how the game behaves: mob AI, drop tables, world rules, player mechanics. The goal is a world that plays differently depending on which state you are in.

Who made it?

FLIPSWITCH is an independent project. All mechanics are built using vanilla Minecraft's datapack and command system — no external tooling, no modding APIs.

Current status

In development. A demo is planned. Follow the news section for updates.

Most likely not. Adding multiplayer functionality would bring a lot more issues. But we'll see.
